# Running the app
Clone the repository:
```bash
git clone https://github.com/Vitin-19/Testing_CORS_Atv.git
cd Testing_CORS_Atv
```

Install the dependencies:
```bash
cd backend
npm i
cd ../frontend
npm i
cd ..
```

Start the backend server:
```bash
cd backend
node server.js
```

Start the frontend:
```bash
cd frontend
npm run serve
```
The server will run on http://localhost:8080 <br>
The frontend will run on http://localhost:3000 <br>
